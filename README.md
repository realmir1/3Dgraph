# 3D Surface and 2D Histogram Plotter :

This project is a Python-based application that visualizes mathematical surfaces in 3D and creates 2D histograms using customizable colormaps. It offers an interactive interface for users to input parameters, define axes, and choose color palettes, enabling flexible data visualization.

<p align="center">
  <img src="https://www.thinkingondata.com/wp-content/uploads/2018/06/Palette_viridis.png" width="500"/>
  <img src="https://miro.medium.com/v2/resize:fit:575/1*J0VmAEUL0zXGvuSICg700Q.png" width="500"/>
</p>


## Features

- **3D Surface Plotting**:
  - Users can input mathematical equations for surfaces.
  - Fully customizable axes labels and ranges.
  - Supports a variety of colormap options to enhance visualization.

- **2D Histogram Plotting**:
  - Generates histograms from random data samples.
  - Displays frequency distribution with user-defined colormaps.

- **Interactive Inputs**:
  - Prompts for user-friendly input of parameters like colormap, axis ranges, and surface equations.

- **Colormap Options**:
  - **Viridis**: Strong, non-distracting, and color-blind-friendly.
  - **Plasma**: Vibrant and high-contrast colors.
  - **Inferno**: Warm tones, ranging from bright yellow to deep red.
  - **Magma**: Softer, darker tones.
  - **Cividis**: Optimized for users with vision impairments.
  - **Rainbow**: A colorful spectrum (not ideal for scientific use).

# 3D Yüzey ve 2D Histogram Çizici:

Bu proje, matematiksel yüzeyleri 3D olarak görselleştiren ve özelleştirilebilir renk haritalarını kullanarak 2D histogramlar oluşturan Python tabanlı bir uygulamadır. Kullanıcıların parametreleri girmeleri, eksenleri tanımlamaları ve renk paletleri seçmeleri için etkileşimli bir arayüz sunarak esnek veri görselleştirmesi sağlar.

<p align="center">

<img src="https://www.thinkingondata.com/wp-content/uploads/2018/06/Palette_viridis.png" width="500"/>

<img src="https://miro.medium.com/v2/resize:fit:575/1*J0VmAEUL0zXGvuSICg700Q.png" width="500"/>

</p>

## Özellikler

- **3D Yüzey Çizimi**:
  - Kullanıcılar yüzeyler için matematiksel denklemler girebilirler.
  - Tamamen özelleştirilebilir eksen etiketleri ve aralıkları.
  - Görselleştirmeyi geliştirmek için çeşitli renk haritası seçeneklerini destekler.
- **2D Histogram Çizimi**:

  - Rastgele veri örneklerinden histogramlar oluşturur.
  - Kullanıcı tanımlı renk haritalarıyla frekans dağılımını görüntüler.
- **Etkileşimli Girişler**:
  - Renk haritası, eksen aralıkları ve yüzey denklemleri gibi parametrelerin kullanıcı dostu girişi için istemler.
- **Renk Haritası Seçenekleri**:
  - **Viridis**: Güçlü, dikkat dağıtıcı olmayan ve renk körü dostu.
  - **Plazma**: Canlı ve yüksek kontrastlı renkler.
  - **Inferno**: Parlak sarıdan koyu kırmızıya kadar değişen sıcak tonlar.
  - **Magma**: Daha yumuşak, daha koyu tonlar.
  - **Cividis**: Görme bozukluğu olan kullanıcılar için optimize edilmiştir.
  - **Gökkuşağı**: Renkli bir spektrum (bilimsel kullanım için ideal değildir).
